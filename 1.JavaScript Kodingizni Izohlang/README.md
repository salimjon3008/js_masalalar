```js
Izohlar-bu JavaScript-ni qasddan e'tiborsiz qoldiradigan kod satrlari. Sharhlar-bu o'zingizga va keyinchalik ushbu kod nima qilishini aniqlashi kerak bo'lgan boshqa odamlarga eslatma qoldirishning ajoyib usuli.

JavaScript-da sharhlar yozishning ikki yo'li mavjud:

JavaScript-//ni joriy satrdagi matnning qolgan qismini e'tiborsiz qoldirishini aytadi. Bu bir in-line fikr emas:

// This is an in-line comment.


Siz bilan boshlanadigan va bilan tugaydigan bir nechta chiziq fikr qilish mumkin /**/. Bu ko'p qatorli sharh:

/* This is a
multi-line comment */


Eslatma: agar kod yozish kabi, siz muntazam ravishda kod qismlari vazifasini oydinlik kiritish uchun izoh kiritish kerak. Yaxshi sharhlash sizning kodingizning niyatini etkazishga yordam beradi-boshqalar uchun ham, kelajakdagi o'zingiz uchun ham.

Har bir sharh turidan birini yaratishga harakat qiling.

```
